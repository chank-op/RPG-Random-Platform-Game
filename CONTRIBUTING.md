# Contributing to RPG (Random Platform Game)

Thank you for your interest in contributing to this project!

## Important Notice

**I did not create this game's code.** This is entirely the work of the original author of **RPG_Game_debug**. The original creator was unable to post this themselves, so I am hosting it on their behalf to make it available to the community.

All credit for the code, game design, and mechanics goes to the original author.

## How to Contribute

1. Fork the repository
2. Create a new branch for your feature (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test your changes in GameMaker Studio 2
5. Commit your changes (`git commit -m "Add your feature"`)
6. Push to your branch (`git push origin feature/your-feature`)
7. Open a Pull Request

## Guidelines

- Keep changes focused and minimal
- Test your changes before submitting
- Follow the existing code style
- Be respectful in discussions

## Reporting Issues

If you find a bug or have a suggestion:
1. Check if the issue already exists
2. Create a new issue with a clear description
3. Include steps to reproduce (for bugs)

## Contact

For questions, open an issue on GitHub.
